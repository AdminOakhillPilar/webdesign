Les enseñamos a los chicos y chicas a leer y escribir de forma divertida, creativa e innovadora. Pero para poder llevar esto a cabo necesitan de una buena nutricion. Por eso el comedor Conin esta dispuesto a brindarles la nutricion y educacion que necesitan para su crecimiento.

Que podemos hacer para ayudar a los niños?

Tips/Metodos para realizarlo

  • Brindar alimentos no perecederos para las bolsas de comida que se entregan semanalmente a las familias que acompañan
  • Hacer recetarios de comida mas atractivos
  • Traer libros de cuentos con dibujos entretenidos
